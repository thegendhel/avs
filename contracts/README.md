## Veritrade Contracts

## Usage

### Build

```shell
$ forge build
```

### Deploy Eigen

```shell
$ forge script script/DeployEigenLayerCore.s.sol --rpc-url <RPC_URL> --broadcast
```

### Deploy Veritrade

```shell
$ forge script script/VeritradepDeployer.s.sol --rpc-url  <RPC_URL> --broadcast
```
